# 🛸 Space Invaders
A classic Space Invaders arcade game remake built with Python and Pygame.

Focused on:
- object-oriented programming
- clean code principles

---

## 🎮 Features
- Arcade-style gameplay
- Progressive enemy challenge system
- Lightweight game loop implementation

---
